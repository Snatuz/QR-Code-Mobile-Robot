FUNÇÃO DE INICIALIZAÇÃO:

modelo de função: int gpioInitialise(void)

a função inicializa a biblioteca. Ela retorna PI_INIT_FAILED caso
não funcione, aliás é uma ótima pratica usar a saída da função para checar
se tudo deu certo.


FUNÇÃO DE CONFIGURAÇÃO DE PINO GPIO:

modelo de função: int gpioSetMode(unsigned gpio, unsigned mode)

unsigned gpio: isso depende do modelo do seu raspberry pi.

modes: podem ser escolhidas pelos seguintes macros:

PI_INPUT = configura o pino como entrada
PI_OUTPUT = configura o pino como saída
PI_ALT = configura pino para dados do tipo UART ou SPI. 

configura um pino para um determinado modo, assim você pode usar ele
para recepção de dados, envio de dados em um configuração específica.


FUNÇÃO DE FINALIZAÇÃO:

modelo de função: void gpioTerminate(void)


JAMAIS esqueça de usar ele no final do programa, se não os recursos
usados pelo programa para o controle dos pinos (memória, threads, channels)
não serão liberados e isso pode gerar problemas.


FUNÇÃO PARA LEITURA DE PINO:

modelo de função : int gpioRead(unsigned gpio)

Lê o sinal que está sendo enviando pelo ou recebido pelo pino.


FUNÇÃO PARA ESCRITA:

modelo de função : int gpioWrite(unsigned gpio, unsigned level) 

Altera o sinal enviado pelo pino para alta ou baixo
